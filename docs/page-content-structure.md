# Lonestar Agro — Page-by-Page Content Structure

**Purpose:** Define sections, headings, and content blocks for every page.

---

## HOME (/)

| Section | Purpose | Content elements |
|---------|---------|------------------|
| Hero | First impression, value prop | Headline, subhead, primary CTA (Request Quote), secondary CTA (Contact), background (premium imagery) |
| Trust bar | Immediate credibility | Logos, certifications, “Serving Texas since [year]”, safety stats |
| Services overview | Segment offerings | 3–6 service cards with short copy + link to service page |
| Why Lonestar | Differentiators | 3–4 bullets: operational excellence, safety, scale, Texas roots |
| Portfolio teaser | Proof | 2–3 featured case studies, “View all” CTA |
| Sustainability | Values | Short statement + link to /sustainability |
| Safety & Compliance | Institutional trust | One block + link to /safety-compliance |
| CTA block | Conversion | “Ready to discuss your project?” + Request Quote + Contact |
| Footer | Navigation + contact | Links, contact, WhatsApp, legal |

## ABOUT (/about)

| Section | Purpose | Content elements |
|---------|---------|------------------|
| Hero | Page title | “About Lonestar Agro” + short tagline |
| Story | Company narrative | Origin, mission, growth, Texas identity |
| Values | Culture & principles | 4–5 value statements (safety, integrity, excellence, stewardship, community) |
| Leadership (optional) | Faces behind the brand | Names, titles, short bios, photos |
| By the numbers | Scale | Stats: years, projects, acres, team size, regions |
| CTA | Next step | “Partner with us” → Request Quote / Contact |
| Footer | Same as site-wide |

## SERVICE PAGE TEMPLATE (/services/[slug])

| Section | Purpose | Content elements |
|---------|---------|------------------|

| Benefits | Why choose us | Bullets or short blocks (efficiency, compliance, expertise) |
| Process / approach | How we deliver | Steps or phases with short copy |
| Applications | Where it’s used | Sectors: municipal, energy, agriculture, development |
